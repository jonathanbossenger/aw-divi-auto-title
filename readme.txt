Auto update Divi Module titles

== Description ==

Divi Auto Title automatically updates the Admin Label in a bunch of Divi modules, based on the most logical field, usually the the Title field

Modules Supported : Audio, Blurb, Button - Button Text, Call To Action, Circle Counter, Contact Form, Countdown Timer, Email Optin, Image - Image Title Text,
Number counter, Person - Name, Testimonial - Author Name, Toggle, Fullwidth Header, Fullwidth Image - Image Title Text, Fullwidth Portfolio

== Installation ==

1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
